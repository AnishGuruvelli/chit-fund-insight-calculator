## Out of scope for V3.0

These are intentionally deferred to keep the release focused:

- Full multilingual UI (Hindi, Telugu, Tamil) beyond formal vs Chitti lingo
- Share-as-generated-image / OG cards

## Implementation order (executed)

1. **Documentation** — This file defines scope and acceptance.
2. **Foundation** — `ThemeProvider`, manifest, PWA plugin, `index.html` meta.
3. **Entry experience** — Splash skip, onboarding overlay, theme toggle on the home hero.
4. **Calculator** — Lingo state, form presets, validation UX, reset/example.
5. **Results & charts** — Explainer, animation, tips, chart sizing, comparison footnote.
6. **History & share** — History UX, clipboard share.
7. **Polish** — Dark-mode classes on main layout; final doc check.

## Acceptance notes

- All calculation and XIRR logic is unchanged; V3 is UX and presentation only.
- Data remains local (localStorage); no new cloud dependencies.
- Offline: service worker precaches build assets; first load still requires network once unless fully cached.
